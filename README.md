# libretro-database-gametdb

Generates DAT files from [GameTDB](http://www.gametdb.com) for [libretro-database](https://github.com/libretro/libretro-database)'s GameCube and Wii entries.

## Usage

1. Install [Node.js](https://nodejs.org)
2. Run `git submodule update --init`
3. Run `npm i`
4. Run `npm test`
5. See the changes in libretro-database
